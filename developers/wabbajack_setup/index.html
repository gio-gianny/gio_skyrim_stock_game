    Wabbajack Compilation Setup

    Compile a new list

    • launch Wabbajack
    • click Create a list
    • click Compile a new list
    • select profiles/Default/modlist.txt
    • set the various compilation info as described in Compilation Info section
    • ignore some files and folders as described in Special Files and Folders

    Compilation info

    Property Value
    Author Gio
    Version 0.0.1
    Title Gio Scratch (Stock Game)
