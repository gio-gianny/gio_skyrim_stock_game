# CS Particle Patch

```project_info
homepage: https://www.nexusmods.com/skyrimspecialedition/mods/147270
about: Fixes lighting on water effects, smoke particles, cloud meshes, and more!
requirements:
    - Community Shaders
    - Enhanced Volumetric Lighting and Shadows (EVLaS) (Optional, for lighting to make sense - just use it)
    - ERM - Enhanced Rocks and Mountains (Optional)
    - Light Limit Fix
    - Mists of Tamriel (Optional)
    - Natural Waterfalls (Optional)
    - Static Mesh Improvement Mod - SMIM (Optional)
    - Terrain Shadows
downloads:
    - CS Particle Patch
```
