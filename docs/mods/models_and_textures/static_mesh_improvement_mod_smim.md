# Static Mesh Improvement Mod - SMIM

```project_info
homepage: https://www.nexusmods.com/skyrimspecialedition/mods/659
about: A massive project to greatly improve the appearance of countless static 3D models in Skyrim. Basically, this is my attempt to make the Skyrim architecture, clutter, furniture, and landscaping much nicer.
requirements:
downloads:
    - SMIM SE 2-08
    - SMIM SE 'Farming Creation Club' Patch
```

## FOMOD settings

```fomod_settings
pages:
    - name: Main Installer Choice
      sections:
        - name: Main Installer Choice
          default:
            value: "Skyrim 2016 Special Edition: Everything"
            notes: All improved meshes will be installed using SMIM's high resolution textures.
```
