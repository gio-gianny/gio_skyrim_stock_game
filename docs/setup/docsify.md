# docsify

## Install dependencies

> [!NOTE]
> You need to restart the shell after installing `nodejs-lts` for `npm` to be available.

```bash
scoop install nodejs-lts
```

## Install docsify

```bash
npm i docsify-cli -g
```

## Create the basic docsify setup

```bash
docsify init ./docs/web
```

## Preview the docs

```bash
docsify serve docs/web/
```
